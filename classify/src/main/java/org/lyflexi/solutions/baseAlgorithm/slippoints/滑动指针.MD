## 滑动窗口
在滑动窗口计算过程中, 我们习惯于**枚举窗口右端点`nums[r] r==i`**, 有以下两个数学规律
- 此时窗口内连续子数组个数`[l,r], [l+1,r]...[r,r]`之和恰好等于窗口长度`r-l+1`
- 此时窗口左侧连续子数组个数`[l-1,r], [l-2,r]...[0,r]`之和恰好等于`l`, 其中`l-1`不在窗口中

不要在内循环中遍历窗口计算, 这将会是O(nk)复杂度, 其中k为窗口大小. 滑动窗口的本质是`O(1)`计算入`compute(nums[i]){...}`以及计算出`compute(nums[l]){...l++;}`

在正式枚举窗口右端点之前, 需要定义以下变量:
- helper, 辅助变量或者辅助数组, 动态存储当前窗口下的计算状态
- ans, 迭代最终答案
- l, 窗口左指针

### 定长窗口模板
以LC1456为例, 定长子串中元音的最大数目
- 计算入: 对每个枚举值(窗口右端点)计算
- 计算出: 当窗口达到固定长度, 迭代最大ans, 同时计算并收缩左指针
```java
    public int maxVowels(String s, int k) {
        int rangeCnt = 0;
        int ans = 0;//求最大
        int l = 0;
        for(int i= 0; i< s.length(); i++){
            //计算入
            if(checkYuanyin(s.charAt(i))){
                rangeCnt++;
            }
            //计算出
            if(i - l +1  == k){
                ans = Math.max(ans, rangeCnt);
                if(checkYuanyin(s.charAt(l))){
                    rangeCnt--;
                }
                l++;
            }
        }

        return ans;
    }

    private boolean checkYuanyin(char c){
        return c == 'a' || c=='e' || c=='i' || c=='o' || c=='u';
    }
```


### 不定长窗口求长度, 越短越合法
以LC03为例, 无重复字符的最长子串
- 计算入: 对每个枚举值(窗口右端点)计算
- 计算出: 随着外层枚举`nums[i]`, 内循环while条件为不符合题意, 计算并收缩左指针
- 迭代答案: 获得一个符合要求的窗口, 于while循环外迭代最大答案
```java
    public int lengthOfLongestSubstring(String s) {
        //s 由英文字母、数字、符号和空格组成, 总共是128个阿斯克码
        int[] dict = new int[128];
        int l = 0;
        int ans = 0;//求最大长度
        for(int i = 0; i<s.length(); i++){
            //注意ascii码的最小值是空格'space'十进制是32, 而不是'a'十进制是97
            dict[s.charAt(i) - ' '] +=1;
            //重复元素一定是右边新加入导致的, 此时需要收缩左边界. 否则可以跳过while直接求max
            while(dict[s.charAt(i) - ' '] > 1){
                dict[s.charAt(l) - ' ']--;
                l++;
            }
            ans = Math.max(ans, i-l+1);
        }
        return ans;
    }
```
### 不定长窗口求长度, 越长越合法
以LC209为例, 长度最小的子数组
- 计算入: 对每个枚举值(窗口右端点)计算
- 计算出: 随着外层枚举`nums[i]`, 内循环while为符合题意, 立即迭代最小答案, 同时计算并收缩左指针
```java
    public int minSubArrayLen(int target, int[] nums) {
        int sum = 0;
        int ans = nums.length+1;//求最小值
        int l = 0;
        //虽然是二重循环, 但由于滑动窗口, 内层循环平均移动1次, 因此最终时间复杂度为n*1 = O(n)
        for(int i = 0; i< nums.length; i ++){
            sum +=nums[i];
            while(sum>=target){
                ans = Math.min(ans, i-l+1);
                sum-=nums[l];
                l++;
            }
        }
    
        return ans==nums.length+1? 0:ans;
    }
```

### 不定长窗口求个数, 越短越合法
以LC713为例, 乘积小于 K 的子数组
- 计算入: 对每个枚举值(窗口右端点)计算
- 计算出: 随着外层枚举`nums[i]`, 内循环while不符合题意, 计算并收缩左指针
- 迭代答案: 获得一个符合要求的窗口, 于while循环外迭代最大答案, 个数恰好为窗口大小i-l+1
```java
    public int numSubarrayProductLessThanK(int[] nums, int k) {
        //1 <= nums[i] <= 1000
        if(k<=1){
            return 0;
        }
        int product = 1;
        int l= 0;
        int cnt = 0;
        //滑动窗口, 以右端点为终点
        for(int i = 0; i<nums.length; i++){
            product*=nums[i];
            while(product>=k){
                product/=nums[l];
                l++;
            }
            //右端点为终点, 当乘积严格小于k的时候, 连续子数组的个数为当前序列长度
            cnt+=i-l+1;
        }

        return cnt;
    }
```

### 不定长窗口求个数, 越长越合法
LC1358, 包含所有三种字符的子字符串数目
- 计算入: 对每个枚举值(窗口右端点)计算
- 计算出: 随着外层枚举`nums[i]`, 内循环while为符合题意, 计算并收缩左指针
- 迭代答案: 获得一个不符合要求的窗口, 于while循环外迭代最大答案, 符合的个数恰好为当前窗口左侧的长度0...l-1, 答案为l
```java
    public int numberOfSubstrings(String s) {
        //辅助数组, 代表子串a,b,c字符的个数
        int[] helper = new int[3];
        int ans = 0;
        int l = 0;
        for(int i = 0; i<s.length(); i++){
            helper[s.charAt(i) - 'a']++;
            //滑动窗口求个数, 越长越合法
            while(helper[0]>0 && helper[1]>0 &&helper[2]>0){
                helper[s.charAt(l) - 'a']--;
                l++;
            }
            ans += l;
        }

        return ans;
    }
```
