对于原数组 `a`，定义其差分数组`（difference array）`为
```text
d[i]=a[0], i==0
d[i]=a[i] − a[i−1], i≥1
```

有差分数组 <-> 原数组 <-> 前缀和数组, 这类似于导函数<->原函数<->积分的关系, 因此求差分、前缀和是一对互逆的过程
- 差分->原数组：对差分求前缀和
- 原数组->前缀和：对原数组求前缀和
- 前缀和->原数组：对前缀和求差分
- 原数组->差分：对原数组求差分
差分数组只关心各区间`interval`的起点和终点, 其余点位用默认值`0`即可, 最后用差分数组求前缀和即可得到原数组, `0`值不影响还原计算

注意, 没有要求差分数组与原数组的长度相等, 你首先需要做的是确定差分数组的长度

实际上差分数组还原出的原数组我们叫做**覆盖次数矩阵**, 要注意区分覆盖次数矩阵和题目给出的初始矩阵不是一个东西

## 性质1: 差分数组求和还原原数组
见LC1094. 拼车

性质 1：从左到右累加 `d` 中的元素，可以得到原数组 `a` 中此刻的值, 即原数组中一个点的值代表差分数组求和到此。
```java
    //差分数组
    public boolean carPooling(int[][] trips, int capacity) {
        //1. 确定差分数组的长度, 题目中给定的0 <= fromi < toi <= 1000
        int N = 1001;
        int[] diff = new int[1001];
        
        //2. 计算差分数组
        //差分数组只关注from和to点位, 其他点位无需计算因为是保持不变的
        // 由于差分数组的用处是累加还原回原数组, 因此非from和to的点位用0表示车内状态没有变化
        for(int[] trip: trips){
            int x = trip[0], from = trip[1], to = trip[2];
            diff[from] += x;
            diff[to] -=x;
        }
        
        //3. 差分数组求和, 还原原数组
        int s = 0;//空间优化, 不需要完全存储原数组
        for(int x: diff){
            s+=x;
            if(s > capacity){
                return false;
            }
        }

        return true;
    }
```

## 性质2: 唯二差分值快速完成原子数组操作
见LC2848. 与车相交的点

性质 2 对于差分数组和原数组来说, 如下两个操作是等价的:

把原数组 `a` 的子数组 `a[i],a[i+1],…,a[j]` 都加上 `x`。

等价于

把差分值 `d[i]` 增加 `x`，把差分值 `d[j+1]` 减少 x。

利用性质 2，我们仅需要通过修改两个差分值`d[i]` 和 `d[j+1]`, 即可映射原子数组的修改, 这只需要 `O(1)` 的时间。

最后再利用性质 1 从差分数组复原出数组 a。
```java
    public int numberOfPoints(List<List<Integer>> nums) {
        //1. 确定差分数组大小
        int maxEnd = 0;
        for(List<Integer> interval: nums){
            maxEnd = Math.max(maxEnd, interval.get(1));
        }
        int len = maxEnd+1;
        int[] diff = new int[len+1];//再加1是因为下面要用到差分数组性质2, 唯二差分值中第二个值是j+1;

        //2. 计算差分数组
        for(List<Integer> interval: nums){
            diff[interval.get(0)]++;
            diff[interval.get(1) + 1]--;
        }


        //3. 还原原数组
        int s = 0, ans = 0;
        for(int x: diff){
            s+=x;
            if(s>0){
                ans++;
            }
        }
        return ans;
    }
```

这里的技巧是我们定义差分数组长度为`len+1`, 多出来的这个位置我们用来表示`d[j+1]`, 防止当`j=len-1`时, `d[j+1]`越界. 

多出来的差分索引这个只是尾部哨兵位, 最终在还原覆盖数组的时候只要我们控制在`[0, len)`, 就不会用到哨兵位, 不对结果造成影响

虽然本题目中在还原覆盖数组的时候没有控制在`[0, len)`, 但是哨兵位的存在也不会影响结果

## 二维差分数组
见LC2132. 用邮票贴满网格图 两个关键点
1. 如何判断邮票可放, 这用二维前缀和为0判断, 如果可放, 批量更新覆盖次数矩阵visit[][]中与被贴住范围对应的格子
2. 如何标记格子被放, 这用二维差分性质2快速还原覆盖次数矩阵中每个格子+1操作, 这可以省略覆盖次数矩阵visit[][]

最后通过二维差分数组还原覆盖次数矩阵, 如果还原过程中存在差分子和为0, 则返回false

最终还原覆盖次数矩阵的过程中, 可以借鉴一维差分还原原数组那样做空间优化, 不必还原出整个覆盖次数矩阵
```java
    //二维差分题目, 还原覆盖次数矩阵的时候空间优化
    public boolean possibleToStamp(int[][] grid, int stampHeight, int stampWidth) {
        //两个关键点
        //1. 如何判断邮票可放, 这用二维前缀和为0判断, 如果可放, 批量更新覆盖次数矩阵visit[][]中与被贴住范围对应的格子
        //2. 如何标记格子被放, 这用二维差分性质2快速还原覆盖次数矩阵中每个格子+1操作, 这可以省略覆盖次数矩阵visit[][]
        //最后通过二维差分数组还原覆盖次数矩阵, 如果还原过程中存在差分子和为0, 则返回false

        int m = grid.length, n = grid[0].length;
        int[][] preS = new int[m+1][n+1];
        for(int i = 0; i<m; i++){
            for(int j = 0; j<n; j++){
                preS[i+1][j+1] = preS[i][j+1] + preS[i+1][j] - preS[i][j] + grid[i][j];
            }
        }

        //定义差分数组长度, 因为要用到性质2, 所以长度都要加1
        int[][] diff = new int[m+1][n+1];
        for(int i = stampHeight-1; i<m; i++){//枚举邮票右下角
            for(int j = stampWidth-1; j<n; j++){
                int i0 = i - stampHeight + 1;
                int j0 = j - stampWidth + 1;
                int subS = preS[i+1][j+1] - preS[i+1][j0] - preS[i0][j+1] + preS[i0][j0];
                if(subS == 0){
                    //维护差分数组的四个顶点
                    diff[i0][j0]++;
                    diff[i0][j+1]--;
                    diff[i+1][j0]--;
                    diff[i+1][j+1]++;
                }
            }
        }

        //二维差分数组求和空间优化
        //以右下角(>0, >0)作为顶点的前缀和, 都依赖左 上 左上, 因此二维差分数组也可以就地求和
        //那么第一列和第一行怎么办? 当作一维数组处理
        for(int i = 0; i<m; i++){
            for(int j = 0; j<n; j++){
                // 在二维差分求和过程中, 我们可以用差分和替换差分值, 这样就可以利用历史求和数据计算新的差分和, 这不影响新的差分和计算
                //此时 diff 已经不再是差分数组, 而是恢复后的覆盖次数矩阵
                if(i > 0) diff[i][j] += diff[i-1][j];

                if(j > 0) diff[i][j] += diff[i][j-1];

                if(i > 0 && j > 0)
                    diff[i][j] -= diff[i-1][j-1];
                if(diff[i][j] == 0 && grid[i][j] == 0){
                    return false;
                }
            }
        }
        return true;
    }
```
