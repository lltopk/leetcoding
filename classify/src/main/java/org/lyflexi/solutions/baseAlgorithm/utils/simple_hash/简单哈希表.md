哈希常见尽量用int[]数组足以, 因为数组的本质也是字典, 并支持随机访问

## HashMap
当题目背景要求元素不重复, 用int[]数组来统计不重复元素个数和 就显得捉襟见肘了, 起码要再遍历一次`int[]`来统计值不为`0`的索引个数

这时利用HashMap.size()可以很方便的得到不重复元素个数和

哈希绝大部分常见中, K为字符, V用来计数或者集合
- V视作计数Count
- V视作集合ArrayList

### 数值V默认值计算
Map<Integer, Integer> helper = new HashMap<>();
```java
if(helper.get(arr[i]) == null){
    helper.put(arr[i], 1);
}else{
    helper.put(arr[i], helper.get(arr[i])+ 1);
}
```

getOrDefault函数简化
```java
helper.put(arr[i], helper.getOrDefault(arr[i], 0) + 1);
```

merge函数传入加法接口:
```java
//map.merge, 当key不存在则初始化value为newV, 当key存在则oldV+newV
// 其中第二个参数是newV
// 其中第三个参数是BiFunction remappingFunction
helper.merge(arr[i], 1, (a,b)-> {return a+b;});
//or
helper.merge(arr[i], 1, Integer::sum);
//merge返回结果是remappingFunction之后的值
```

### 集合V默认值计算
Map<Integer, List<Integer>> helper = new HashMap<>();

由于集合V执行add返回布尔值而不是集合本身, 因此getOrDefault只能初始化集合V, 无法直接给集合V追加元素

所以集合V场景下使用computeIfAbsent完成简化操作
```java
//computeIfAbsent返回对象V
helper.computeIfAbsent(1, (key)-> {return new ArrayList<>();}).add(1);
```

## 一维数组int[]
对于字符场景, 我们知道ascii集总共有128个字符, 十进制从0~127. 此时可以用数组索引表示特定字符
- 最小字符是NULL, 不可见, 但可以表示为转义序列'\0'
- 最大字符是DEL, 不可见, 无法表示

其中**可见字符位于ascii表中间共有95个**, 包括英文字母、数字、符号和空格, 十进制从32~126
- 最小字符表示是'space'
- 'A'十进制是65
- 'a'十进制是97
- 最大值是'~'十进制为126

代码实现如下
```java
//字符char为整个ascii集, 共计128个, 将char - '\0'作为索引
int[] dict = new int[128];
//字符char为整个可见字符集, 共计95个, 将char - '空格'作为索引
int[] dict = new int[95];
//字符char为小写a~z, 共计26个, 将char - 'a'作为索引
int[] dict = new int[26];
```

以上顶多初始化95个可见字符的字典足以解题

## 二维数组`int[][2]`
如果一维数组`int[]`索引无法表示key的时候, 就要用到二维数组了`int[][2]`;

对于二维数组`int[][2]`来说, 此时`int[i]`表示第几个`Entry`, `int[i][0]`表示该`Entry`的`key`, `int[i][1]`表示该`Entry`的`value`

例如LC853. 车队, 输入：`target = 12, position = [10,8,0,5,3], speed = [2,4,1,1,3]`

```java
class Solution {
    /**
     * 单调栈
     * @param target
     * @param position
     * @param speed
     * @return
     */
    public int carFleet(int target, int[] position, int[] speed) {
        //new map
        int[][] cars = new int[position.length][2];
        for(int i = 0; i< position.length; i++){
            cars[i][0] = position[i];
            cars[i][1] = speed[i];
        }
        //根据车的位置排序
        Arrays.sort(cars, (a, b) -> { return a[0] - b[0];});
        //单调栈, 存车辆假设独自到达target的时间
        Deque<Double> sk = new ArrayDeque<>();
        int ret = position.length;
        for(int i= position.length - 1; i>=0; i--){
            Double t = (double)(target - cars[i][0]) / cars[i][1];
            //合并车队, 注意sk.peek() == t的时候也满足车队要求, 会在target相遇
            while(! sk.isEmpty() && sk.peek() < t){
                sk.pop();
            }
            if(! sk.isEmpty()){
                //有效车队的右侧不存在更大时间, 因此对right[]数组中的0值求和
                ret--;
            }
            //假设cars[i][0]< cars[j][0], 那么car[i]追击成功需满足ti < tj 
            sk.push(t);
        }
        return ret;
    }
}
```
