
## 双变量问题 枚举右维护左
_对于 双变量问题，例如两数之和 `nums[i] + nums[j] == target`_

可以枚举右边的`nums[r]`, 这可以转换成 单变量问题，也就是在 nums[r]左边查找是否有`nums[l] = target - nums[r]`

上述枚举过程可能需要辅助变量`val`, 辅助哈希表`HashMap`, 或者当字典大小确定的时候用辅助数组`int[]/int[][]`维护。 

我把这个技巧叫做 **枚举右，维护左**. 具体可以拆分为两类场景
- 当右是影响因子, 我们要后更新辅助结构, 这类场景辅助结构用于存储计数值
- 当左是影响因子, 我们要先更新辅助结构, 这类场景辅助结构用于存储滚动值

### 辅助结构存储计数值, 后更新辅助结构
_辅助结构用于存储计数值, 这类场景要先更新答案后更新辅助结构，辅助结构在任意计算轮次中都不应该包含当前枚举右, 否则会导致额外错误的计算_

例如LC624. 数组列表中的最大距离, 利用辅助变量
```java
    public int maxDistance(List<List<Integer>> arrays) {
    //怎么利用各个数组有序的条件呢？  可以直接O(1)访问每个数组的第一个元素为局部最小值， 每个数组的最后一个元素为局部最大值
    int ans = 0;
    int preMin = Integer.MAX_VALUE/2;//防止减法溢出
    int preMax = Integer.MIN_VALUE/2;//题目元素中存在负数
    int n = arrays.size();
    for(int r = 0; r< n; r++){
        List<Integer> list = arrays.get(r);
        //假设遍历到右端点, 当前最大差有两种算法
        //1. preMax - list.get(0)
        //2. list.get(list.size()-1) - preMin
        ans = Math.max(ans, Math.max(preMax - list.get(0), list.get(list.size()-1) - preMin));
        preMin = Math.min(preMin, list.get(0));
        preMax = Math.max(preMax, list.get(list.size() - 1));
    }

    //不能最后return preMax - preMin, 因为有可能最大值和最小值出现在同一个数组， 不符合题意
    return ans;
}
```

例如LC1512. 好数对的数目, 利用辅助哈希表
```java
    public int numIdenticalPairs(int[] nums) {
        int ans = 0, n = nums.length;
        //记录元素出现的次数
        Map<Integer, Integer> map = new HashMap<>();
        for(int r = 0; r<n; r++){
            if(!(map.get(nums[r]) == null)){
                //右端点是影响因子, 每出现一个有效右端点, 好数对的增加恰好等于元素过去出现的次数
                ans+=map.get(nums[r]);
            }
            map.merge(nums[r], 1, Integer::sum);
        }

        return ans;
    }
```

例如2342. 数位和相等数对的最大和, 利用辅助哈希表
```java
    public int maximumSum(int[] nums) {
        int n = nums.length, ans = -1;
        //数位和最多为9个9
        int[] dict = new int[82];//统计数位和 对应 实际元素最大值
        //初始化MIN_VALUE
        for(int s = 0; s <=81; s++){
            dict[s] = Integer.MIN_VALUE;
        }
        for(int r = 0; r< n; r++){
            int sumBit = 0, num = nums[r];
            while(num!=0){
                sumBit+=num%10;
                num/=10;
            }
            //由于初始化了MIN_VALUE， 因此若dict[sumBit]不存在， 恰好也满足ans为-1
            ans = Math.max(ans, nums[r] + dict[sumBit]);
            //维护左
            dict[sumBit] = Math.max(dict[sumBit], nums[r]);
        }

        return ans;
    }
```

### 辅助结构存储滚动值, 先更新辅助结构
_辅助结构存储滚动值, 这类场景中辅助结构取决于当前的右端点, 比如`l = r - mid / l = r - mid +1`, 因此要先更新辅助结构后更新答案_

滚动计算的辅助结构含义是, 以`l`为首的前缀元素的最大值`mx`或者最小值`mn`
```java
mx = f(l)
mn = f(l)
```

这类题目有3584.子序列首尾元素的最大乘积, 维护最大值`mx`/最小值`mn`
```java
    public long maximumProduct(int[] nums, int m) {
        //最左侧的元素为r - (m - 1), 注意由于是子序列而不是子数组, 所以该位置是最小约束, 左界不能在该位置的右侧
        //滚动求该位置的最大值和最小值
        int mn = Integer.MAX_VALUE, mx = Integer.MIN_VALUE;
        long ans = Long.MIN_VALUE;//答案有可能是负值
        int n = nums.length;
        //r从m-1开始, 保证r-(m-1)会从0开始
        for(int r = m-1; r< n; r++){
            int i = r - m + 1;
            mn = Math.min(mn, nums[i]);
            mx = Math.max(mx, nums[i]);
            long y = (long)nums[r];//防止下面乘法越界
            ans = Math.max(ans, Math.max(mn*y, mx*y));
        }
        return ans;
    }
```

2905.找出满足差值条件的下标 II, 维护最大值索引`mxIdx`/最小值索引`mxIdx`
```java
    public int[] findIndices(int[] nums, int indexDifference, int valueDifference) {
        int maxIdx = 0;
        int minIdx = 0;
        for (int j = indexDifference; j < nums.length; j++) {
            //枚举右j, 维护左: j - indexDifference
            int i = j - indexDifference;
            if (nums[i] > nums[maxIdx]) {
                maxIdx = i;
            } else if (nums[i] < nums[minIdx]) {
                minIdx = i;
            }
            //abs(nums[i] - nums[j]) >= valueDifference等价于以下两个场景

            //1. |nums[j] - nums[minIdx]|
            if (Math.abs(nums[j] - nums[minIdx]) >= valueDifference) {
                return new int[]{minIdx, j};
            }
            //2. |nums[maxIdx] - nums[j]|, 如[5,1,4,1], 答案为[0,3]
            if (Math.abs(nums[maxIdx] - nums[j]) >= valueDifference) {
                return new int[]{maxIdx, j};
            }

        }
        return new int[]{-1, -1};
    }
```
