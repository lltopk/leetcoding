
## 栈ArrayDeque
Java中的`Stack`是线程同步工具类, 每个方法都有`synchronized`声明, 并且底层是`Vector`模型, 比较古老, 在大数据量频繁操作的场景下(leetcode)性能会很差

`leetcode`场景下推荐使用`Deque<Integer> stack = new ArrayDeque<>();`, 是专门为栈和队列设计的, 同时支持队列和栈, 性能好
```text
入队: offer(e)->addLast(e)
出队: remove()->removeFirst()

入栈: push(e)->addFirst(e)
出栈: pop()->removeFirst()
```

## 列表/数组/字符串模拟栈
用列表/数组/字符串模拟栈的好处还有就是能够按照自底向上的扫描顺序还原字符串, 见71. 简化路径
```java
    public String simplifyPath(String path) {
        List<String> stack = new ArrayList<>();
        for(String s: path.split("/")){
            if("".equals(s) || ".".equals(s)){
                continue;
            }
            if(!"..".equals(s)){
                stack.add(s);
            }else if(!stack.isEmpty()){
                stack.remove(stack.size()-1);
            }
        }
        return "/" + String.join("/", stack);
    }
```

下面给出每种模拟的实现例题
### 列表模拟
单独创建个列表/数组模拟栈，这样就不局限于栈提供的有限API, 通过数组索引支持随机访问这样性能更高， 见LC1472. 设计浏览器历史记录

与JDK保持一致的是, 我们列表/数组模拟栈的时候也习惯于将栈索引始终保持peek+1的位置, 因此实际栈顶元素是`nums[tail-1]`

如果题目明确栈大小, 你可以用数组模拟, 如果题目不明确栈大小, 你只能用列表模拟
```java
public class LC1472_BrowserHistory2 {
    //用列表/数组模拟栈， 支持随机访问性能更高
    final List<String> history = new ArrayList<>();
    //游标卡尺, tail指向peek+1位置, 因此本题索引范围是[0, tail-1]
    int tail = 0;//和jdk源码一样, tail永远指向实际位置的下个位置
    public LC1472_BrowserHistory2(String homepage) {
        history.add(homepage);
        tail++;
    }

    public void visit(String url) {
        //SubList内部持有的仍然是完整list的引用
        history.subList(tail, history.size()).clear();
        history.add(url);
        tail++;
    }

    public String back(int steps) {
        int cur = Math.max(0, tail-1-steps);
        tail = cur+1;
        return history.get(cur);
    }

    public String forward(int steps) {
        int cur = Math.min(history.size()-1, tail - 1 +steps);
        tail = cur + 1;
        return history.get(cur);
    }
}
```

### 数组模拟
需要注意的是当前下标`i`指的是模拟栈的`tail`索引, `nums[tail] === null`, 而不是题目输入数组的索引, 为了加深对列表/数组模拟栈的理解下面见LC946. 验证栈序列
```java
    public boolean validateStackSequences(int[] pushed, int[] popped) {
        int i = 0, j = 0, n = pushed.length;
        int[] helper = new int[n];
        //i是新数组helper的索引
        for(int num: pushed){
            //模拟栈, 这道题栈索引i始终保持peek+1的位置
            helper[i++] = num;
            while(i>0 && helper[i-1] == popped[j]){
                i--;
                j++;
            }
        }
        return i == 0;
    }
```


### 字符串模拟
`String.join("delimiter", "listRef")`按照相对顺序返回字符串答案的前提是, `listRef`中元素类型必须是`CharSequence`如`String`, 而不能是`Character`

很多题目的输入都是`char[] sa = s.toCharArray();`, 因此有必要单独对`Character`类型的元素题目单独讨论, 我们可以用`StringBuilder`处理这类问题

`StringBuilder`提供了类似于`List`的根据索引随机访问字符的`API`, 见LC1047. 删除字符串中的所有相邻重复项
```java
    public String removeDuplicates(String s) {
        //返回原顺序的字符串, 因此用列表模拟栈最方便了
        StringBuilder stack = new StringBuilder();
        char[] sa = s.toCharArray();
        for(char c: sa){
            if(!stack.isEmpty() && c == stack.charAt(stack.length()-1)){
//                stack.deleteCharAt(stack.length()-1);
                stack.setLength(stack.length() - 1);
                continue;
            }

            stack.append(c);
        }
        return stack.toString();
    }
```
## 索引栈
有些题目栈存储的不是元素本身, 而是元素如`'('`在原始字符串的下标索引

见LC1190. 反转每对括号间的子串
```java
    public String reverseParentheses(String s) {
        //栈仅仅用来存储'('的下标i, 当遇到')'的时候, 将(i, j)中的数据reverse
        Deque<Integer> stack = new ArrayDeque<>();
        char[] sa = s.toCharArray();
        for(int i = 0 ;i <sa.length; i++){
            char c = sa[i];
            if(c =='('){
                stack.push(i);
                continue;
            }

            if(c == ')'){
                reverse(sa, stack.pop()+1, i);
                continue;
            }
        }

        StringBuilder ret = new StringBuilder();
        for(char c: sa){
            if(c!='(' && c!=')'){
                ret.append(c);
            }
        }

        return ret.toString();
    }

    /**
     左闭右开区间
     */
    private void reverse(char[] sa, int start, int end){
        end-=1;
        //偶数情况: 跳出循环恰好start > end
        //奇数情况: 跳出循环恰好start == end
        while(start < end){
            char t = sa[start];
            sa[start] = sa[end];
            sa[end] = t;
            start++;
            end--;
        }
    }
```

见LC1249. 移除无效的括号, 这题很好, 除了用栈存储`'('`的索引之外, 还构造了标记数组, 通过懒计算的方式解决了结果字符串的顺序问题
```java
    public String minRemoveToMakeValid(String s) {
        StringBuilder ret = new StringBuilder();
        //栈仅仅存储'('的索引
        Deque<Integer> stack = new ArrayDeque<>();
        //标记有效位
        int[] mark = new int[s.length()];
        for(int i = 0; i<s.length(); i++){
            char c = s.charAt(i);
            if(stack.isEmpty() && c == ')'){
                continue;
            }

            //有效的'(...)'之间的元素如果已经加入答案, 则无法再后期给答案插入'('
            // 本题目采用先标记法, 最终统一构造答案 , 保证答案字符顺序
            if(!stack.isEmpty() && s.charAt(stack.peek()) == '(' && c ==')'){
                mark[stack.pop()] = 1;
                mark[i] = 1;
                continue;
            }
            if(c == '('){
                stack.push(i);
            }

            if(Character.isLetter(c)){
                mark[i] = 1;
            }
        }

        for(int i = 0; i< s.length(); i++){
            if(mark[i] == 1){
                ret.append(s.charAt(i));
            }
        }
        return ret.toString();
    }
```

## pair栈

### 同字符pair栈
我们要从字符串 `s` 中选择 `k` 个相邻且相等的字母, 这属于同字符连续消除`k`次问题

解题: 栈元素同时记录连续出现次数, 这样可以避免每次回溯统计次数, 降低时间复杂度, 如LC1209.删除字符串中的所有相邻重复项 II
```java
    public String removeDuplicates(String s, int k) {
        //栈元素同时记录连续出现次数, 这样可以避免每次回溯统计次数, 降低时间复杂度
        List<int[]> stack = new ArrayList<>();
        for(char c: s.toCharArray()){
            if(! stack.isEmpty() && stack.get(stack.size() - 1)[0] == c){
                stack.get(stack.size() - 1)[1]++;
                if(stack.get(stack.size() - 1)[1] == k){
                    stack.remove(stack.size() - 1);
                }
                //连续次数不超过k的, 留在栈中即可, 要么等待机会消消乐, 要么永远无法消除就是答案
            }else{
                stack.add(new int[]{c, 1});
            }
        }

        StringBuilder ret = new StringBuilder();
        for(int[] pair: stack){
            for(int i = 0; i< pair[1]; i++){
                ret.append((char)pair[0]);
            }
        }

        return ret.toString();
    }
```

### 字符对pair栈
我们要从字符串 `s` 中移除所有长度为 `k` 的连续括号对, 比如当`k==3`时目标移除对象为`((()))`

解题: 栈元素同时记录连续出现次数, 这样可以将连续`k`次的左右括号抽象为`1`次的左右括号, 否则我们无法同时统计第几次`(`或`)`, 这样可以转化成LC20. 有效的括号

见LC3703. 移除K-平衡子字符串, 具体需要先入栈将`(`和`)`进行计数压缩, 再对抽象数组进行消除, 注意此时枚举的元素已经入栈了
```java
    public String removeSubstring(String s, int k) {
        //栈元素同时记录连续出现次数, 这样可以将k次的左右括号抽象为1次的左右括号, 就变成了LC20. 有效的括号
        List<int[]> stack = new ArrayList<>();

        //栈中只有两种括号, 但长度可不止2, 如()()(), k=2
        for(char c: s.toCharArray()){
            //先入栈, 先进行抽象
            if(!stack.isEmpty() && stack.get(stack.size() - 1)[0] == c){
                stack.get(stack.size() - 1)[1]++;
            }else{
                stack.add(new int[]{c, 1});
            }

            //再对抽象数组进行消除, 注意此时枚举的元素已经入栈了
            if(stack.size()>1 && c == ')' && stack.get(stack.size() - 1)[1] == k && stack.get(stack.size() - 2)[1] >= k){
                stack.remove(stack.size() - 1);//移除)
                stack.get(stack.size() - 1)[1] -= k;//现在栈顶是(, 减去k
                if(stack.get(stack.size() - 1)[1] == 0){
                    stack.remove(stack.size() - 1);//可能移除(
                }

                //连续次数超过k的剩余(, 留在栈中即可, 要么等待机会消消乐, 要么永远无法消除就是答案
            }
        }

        StringBuilder ret = new StringBuilder();
        for(int[] pair: stack){
            for(int i = 0; i< pair[1]; i++){
                ret.append((char)pair[0]);
            }
        }

        return ret.toString();
    }
```
