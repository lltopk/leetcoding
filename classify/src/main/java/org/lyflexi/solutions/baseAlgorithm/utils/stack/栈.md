## 列表模拟栈
单独创建个列表/数组模拟栈， 支持随机访问这样性能更高， 见LC1472. 设计浏览器历史记录
```java
    final List<String> history = new ArrayList<>();
    //游标卡尺
    int cur = 0;
    public LC1472_BrowserHistory2(String homepage) {
        history.add(homepage);
    }

    public void visit(String url) {
        //SubList内部持有的仍然是完整list的引用
        history.subList(++cur, history.size()).clear();
        history.add(url);
    }

    /**
     * 缩回卡尺
     * @param steps
     * @return
     */
    public String back(int steps) {
        cur = Math.max(0, cur-steps);
        return history.get(cur);
    }

    /**
     * 前进卡尺
     * @param steps
     * @return
     */
    public String forward(int steps) {
        cur = Math.min(history.size()-1, cur+steps);
        return history.get(cur);
    }
```

需要注意的是当前下标`cur`指的是模拟栈的索引, 而不是题目输入数组的索引, 为了加深对列表/数组模拟栈的理解下面见LC946. 验证栈序列
```java
    public boolean validateStackSequences(int[] pushed, int[] popped) {
        int i = 0, j = 0, n = pushed.length;
        int[] helper = new int[n];
        //i是新数组helper的索引
        for(int num: pushed){
            //模拟栈, 这道题栈索引i始终保持peek+1的位置, 即左闭右开[0, peek+1)
            helper[i++] = num;
            while(i>0 && helper[i-1] == popped[j]){
                i--;
                j++;
            }
        }
        return i == 0;
    }
```
